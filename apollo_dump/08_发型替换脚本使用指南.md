# 发型替换脚本使用指南

> 本文档说明如何使用 `1.py` 脚本在房间阶段替换角色发型。
> 适用于 FreeStyle 街头篮球游戏。

---

## 一、脚本功能

在房间阶段，通过 hook `MSVCR100.dll` 的 `sprintf` 函数，将源发型的 ItemCode 替换为目标发型的 ItemCode，从而实现发型切换。

**限制**：练习场阶段不生效（练习场有第二条加载路径，绕过 sprintf）。

## 三、发型配置文件格式

`C:\Users\w\Desktop\hair_styles.json` 格式：

```json
[
  {
    "itemCode": 50125711,
    "name": "紫色超赛发型",
    "pak": 768
  },
  {
    "itemCode": 50122451,
    "name": "鬼魅发型",
    "pak": 742
  }
]
```

**字段说明**：
- `itemCode`: 发型的物品代码（必须正确，否则光头）
- `name`: 发型名称（用于显示和搜索）
- `pak`: 资源所在的 pak 编号（对应 `res{pak}.pak` 文件）

## 四、使用方法

```powershell
# 列出所有可用发型
python 1.py list

# 按 ItemCode 切换
python 1.py 50125711      # 切换到紫色超赛

# 不带参数运行（使用默认配置）
python 1.py               # 默认：美丽梦想 → 紫色超赛
```

**运行后命令**：
- `quit` 或 `q` - 退出脚本
- `count` - 查看替换次数
- `list` - 查看发型列表

## 五、增补新发型

### 步骤

1. **确认 ItemCode**：从游戏数据或 `itemshop.json` 查找发型的正确 ItemCode

2. **确认 pak 编号**：该发型资源所在的 pak 文件编号

3. **编辑 JSON 文件**：
   ```json
   {
     "itemCode": 5012xxxx,
     "name": "新发型名称",
     "pak": 7xx
   }
   ```

4. **保存文件**

5. **验证**：运行 `python 1.py list` 查看新发型是否出现

### 注意事项

- **ItemCode 必须正确**：错误的 ItemCode 会导致光头
- **pak 文件必须存在**：游戏目录下需要有对应的 `res{pak}.pak`
- **从可靠来源获取 ItemCode**：建议从 `itemshop.json` 或游戏实际数据提取

## 六、技术原理

### 替换逻辑

```
sprintf("customize\item\i%d.xml", ItemCode)
         ↓
Hook 拦截，修改 args[2] 为目标 ItemCode
         ↓
同时修改栈上的值 [ebp-0xD8]（LoadItemFile 读取这个值）
         ↓
游戏用新 ItemCode 构造路径
         ↓
LoadItemFile 从对应 pak 读取 BML
         ↓
AcquireSMD 加载 SMD 资源
         ↓
显示目标发型
```

### 关键代码

```javascript
// 1. 修改 sprintf 参数
args[2] = ptr(DST_IC);

// 2. 修改栈上的值（关键！否则光头）
var callerEbp = this.context.ebp;
callerEbp.sub(0xD8).writeU32(DST_IC);
```

## 八、常见问题

### Q: 运行后显示光头？

**原因**：
1. ItemCode 错误
2. 对应的 pak 文件不存在
3. 发型资源在 pak 中不存在

**解决**：
1. 确认 ItemCode 正确（从可靠来源获取）
2. 检查游戏目录是否有 `res{pak}.pak`
3. 检查 pak 内是否有该发型的资源文件

### Q: 练习场发型恢复原样？

**原因**：练习场有第二条加载路径，绕过 sprintf，直接加载原始 ItemCode 的资源。

**现状**：目前无法解决，是游戏架构限制。

### Q: 如何获取正确的 ItemCode？

**方法**：
1. 从 `itemshop.json` 数据库查询
2. 从游戏内存扫描
3. 从 pak 文件名推断（`i{ItemCode}_FN.smd`）
