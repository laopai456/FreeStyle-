## 整体布局

### 1. 窗口结构
- **窗口大小**：1280x720，16:9 比例
- **窗口样式**：无边框，透明背景
- **背景**：蓝粉紫渐变背景

### 2. 顶部标题栏
- **位置**：窗口顶部
- **高度**：80px
- **内容**：

  - 右侧：窗口管理按钮（最小化、最大化、关闭），玻璃拟态风格
- **样式**：透明背景，与主窗口融为一体

### 3. 标签页导航
- **位置**：标题栏下方
- **内容**：三个标签页按钮
  - "服装道具"（Clothing Items）
  - "地图切换"（Map Switch）
  - "常用工具"（Common Tools）
- **样式**：玻璃拟态风格，选中状态有高亮效果

### 4. 主内容区域
- **位置**：标签页下方，占据窗口主要空间
- **样式**：玻璃拟态面板，圆角 16px，半透明背景，柔和阴影

## 技术实现提示

- **玻璃拟态效果**：通过 Border 元素的 Background、BorderBrush 和 Effect 属性实现
- **渐变背景**：使用 LinearGradientBrush 实现
- **动画效果**：使用 VisualStateManager 和 Storyboard 实现
- **窗口管理**：通过自定义事件处理实现窗口拖动、最小化、最大化和关闭功能
