   - 添加 `icon` 文件夹，用于存放应用程序图标

3. **构建配置**

## 使用方法

### 更改应用程序图标
1. 打开 `icon` 文件夹
2. 将新的图标文件重命名为 `favicon.ico` 并替换原文件
3. 运行 `dotnet build` 重新构建项目
4. 构建完成后，新的图标将应用到应用程序中

### 注意事项
- 任务栏图标会立即更新为新图标
- 桌面快捷方式和资源管理器中的图标可能需要清除 Windows 图标缓存后才能更新
- 清除图标缓存的方法：
  - 按 `Win + R` 打开运行对话框
  - 输入 `ie4uinit.exe -ClearIconCache` 并按 Enter
  - 或输入 `cmd /c taskkill /f /im explorer.exe && explorer.exe` 重启资源管理器
