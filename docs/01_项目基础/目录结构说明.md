## 核心文件说明

| 文件 | 说明 |
|------|------|
| `UI/Windows/MainWindow.xaml(.cs)` | 主窗体，包含服装变更、地图入口、工具入口、主题应用、视频背景播放 |
| `UI/Windows/SkinWindow.xaml(.cs)` | 皮肤设置窗口，当前负责主题选择/预览/应用；后续动态背景上传也放这里 |
| `Core/Models/SkinTheme.cs` | 主题 JSON 对应的数据模型，支持 `videoPath` |
| `Core/Services/ThemeLoader.cs` | 扫描、保存主题，确保暗黑主题存在 |
| `Core/Services/ThemeApplier.cs` | 应用主题颜色、背景、字体，解析视频完整路径 |
| `Core/Services/SkinManager.cs` | 维护主题列表和当前主题配置 |
| `Core/Config/ConfigService.cs` | 保存游戏目录、窗口尺寸、当前主题等配置 |
| `Core/Utilities/conmon.cs` | 文件复制、命令执行等通用逻辑 |
| `pack/resources.exe` | PAK 文件解包/打包工具 |

## 动态背景相关目录

| 路径 | 说明 |
|------|------|
| `videos/` | 存放所有动态背景视频 |
| `skins/{主题ID}/theme.json` | 主题配置，`window.background.type = video` 时启用视频背景 |
| `bin/Debug/net8.0-windows/videos/` | 运行时实际读取的视频目录 |
| `bin/Debug/net8.0-windows/skins/` | 运行时实际读取的主题目录 |
