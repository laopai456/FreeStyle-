## 主项目目录

```
FS服装搭配专家v5.3.6\
├── FrmMain.cs                  # 主窗体（核心逻辑）
├── ViewOneIconFrm2.cs          # 道具预览窗体
├── OperationDebugger.cs         # 调试监听器
├── conmon.cs                   # 公共工具类
├── Program.cs                  # 程序入口

├── bin\Debug\                  # 编译输出目录
│   ├── logs\                  # 调试日志目录
│   │   └── operation_debug.log
│   └── FS服装搭配专家v5.3.6.exe

├── othercookies\              # 它服解包文件目录

```

## 核心文件说明

| 文件 | 说明 |
|------|------|
| `FrmMain.cs` | 主窗体，包含所有核心逻辑 |
| `ViewOneIconFrm2.cs` | 道具图标预览和设置变更前后 |
| `OperationDebugger.cs` | 操作流程监听和日志记录 |
| `conmon.cs` | 公共工具方法（复制目录、运行命令等） |
| `Program.cs` | 程序入口点 |
