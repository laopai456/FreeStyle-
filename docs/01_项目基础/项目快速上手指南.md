# 项目快速上手指南

## 第一步：理解项目

这是一个 **FreeStyle 街头篮球 WPF/.NET 8 辅助工具**，主要功能：

- 查看并搜索游戏道具、图标、特效
- 将一个道具的外观替换为另一个道具
- 支持地图切换、背景切图、PAK 字节补齐
- 支持主题换肤和 LibVLC 动态视频背景

## 第二步：快速理解代码逻辑

### 核心流程（3句话）

1. **加载数据** → 复制/解包 `item_text.pak` → 解析 `itemshop.txt` → 绑定服装/特效列表
2. **选择变更** → 左键添加到变更前，`Ctrl + 左键` 添加到变更后，支持为目标服装附加特效
3. **执行变更** → 解包 item pak → 复制目标 BML 到源 BML 文件名 → 重打包到游戏目录

### 代码文件优先级

1. **先看** `README.md` - 当前 WPF 架构和动态背景说明
2. **再看** `核心代码文件索引.md` - 快速定位关键类和方法
3. **最后看** `完整操作逻辑梳理.md` / `调试监听器说明.md` - 了解业务流程和日志

## 第三步：运行程序


## 第四步：使用调试日志

程序运行后，关键操作会输出到控制台或 `logs/operation_debug.log`。

查看日志了解：

- 用户点击了什么
- 程序执行了哪些步骤
- 哪个文件复制、解包、打包失败
- 视频背景是否加载成功

## 第五步：修改代码

### 添加新功能

1. 先确认入口窗口：`UI/Windows/MainWindow.xaml(.cs)` 或 `UI/Windows/SkinWindow.xaml(.cs)`
2. 修改 XAML UI
3. 在 `.xaml.cs` 中实现事件逻辑
4. 添加足够日志
5. `dotnet build -c Debug` 编译测试
6. 运行程序做手工验证

### 常见修改场景

- **主界面功能**：改 `UI/Windows/MainWindow.xaml` 和 `MainWindow.xaml.cs`
- **皮肤/动态背景**：改 `SkinWindow.xaml(.cs)`、`SkinTheme.cs`、`ThemeLoader.cs`、`ThemeApplier.cs`
- **背景切图**：改 `BgCropWindow.xaml(.cs)`
- **补充字节**：改 `FillBytesWindow.xaml(.cs)`
- **地图切换**：改 `MapControl.xaml(.cs)`
- **配置路径**：改 `Core/Config/ConfigService.cs` / `AppConfig.cs`

## 第六步：动态视频背景上传待开发

当前底层已支持视频背景，但还没有“一键上传视频作为动态主题”的 UI。

建议实现：

```text
SkinWindow 增加“导入动态背景”按钮
→ 选择本地 mp4
→ 复制到 videos/custom_时间戳.mp4
→ 创建 skins/custom_video_时间戳/theme.json
→ skinManager.LoadThemes()
→ 刷新主题列表
→ 选择并应用新主题
```

关键文件：

- `UI/Windows/SkinWindow.xaml`
- `UI/Windows/SkinWindow.xaml.cs`
- `Core/Models/SkinTheme.cs`
- `Core/Services/ThemeLoader.cs`
- `Core/Services/SkinManager.cs`
