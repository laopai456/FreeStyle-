# 核心代码文件索引

> 当前项目为 WPF/.NET 8 版，主逻辑不再使用旧 WinForms `FrmMain`。

## 主窗口 - `UI/Windows/MainWindow.xaml(.cs)`

| 功能 | 关键成员/方法 | 说明 |
|------|---------------|------|
| 窗口初始化 | `MainWindow()` | 初始化窗口、皮肤管理器、服装数据、窗口尺寸配置 |
| 加载配置 | `InitializeConfig()` | 读取/选择游戏安装目录，检查 `item_text.pak` |
| 加载服装数据 | `InitializeItemList()` / `GetNewItem()` | 后台解包并解析 `itemshop.txt` |
| 搜索服装/特效 | `txtSearch_TextChanged()` / `SearchAsync()` | 200ms 防抖过滤列表 |
| 加载图标 | `btnLoadImg_Click()` / `LoadImages()` | 解包 icon pak 并缓存 PNG |
| 添加变更前 | `lstClothing_SelectionChanged()` / `AddToBeforeList()` | 普通左键选中服装加入变更前 |
| 添加变更后 | `lstClothing_SelectionChanged()` / `AddToAfterList()` | `Ctrl + 左键` 加入变更后 |
| 添加特效 | `lstEffect_SelectionChanged()` / `SelectAfterItemForEffect()` | 先选特效，再 `Alt + 左键` 应用到目标服装 |
| 更新变更区 | `UpdateBeforePanel()` / `UpdateAfterPanel()` | 刷新变更前/后图标面板 |
| 确认变更 | `btnConfirm_Click()` | 校验数量、弹出确认框、逐项执行替换 |
| 核心替换 | `ExecuteClothingChange()` | 复制 BML、重打包 item pak、可写入特效到 `itemshop.txt` |
| 背景切图入口 | `btnBgCrop_Click()` | 打开 `BgCropWindow` |
| 补充字节入口 | `btnFillBytes_Click()` | 打开 `FillBytesWindow` |
| 换肤入口 | `btnSkin_Click()` | 打开 `SkinWindow` |
| 应用主题 | `ApplyCurrentTheme()` | 应用当前 SkinTheme 到主窗口/地图控件 |
| 视频背景 | `ApplyVideoBackground()` | 判断 video 主题并启动/停止 LibVLC 播放 |
| VLC 初始化 | `EnsureVLCInitialized()` | 初始化 LibVLC、设置视频回调 |
| 视频帧渲染 | `OnVideoFormat()` / `OnLockVideo()` / `OnUnlockVideo()` | 解码帧写入 `WriteableBitmap` |
| 窗口尺寸保存 | `LoadWindowSize()` / `MainWindow_SizeChanged()` | 保存主窗口宽高 |

## 皮肤窗口 - `UI/Windows/SkinWindow.xaml(.cs)`

| 功能 | 关键成员/方法 | 说明 |
|------|---------------|------|
| 初始化 | `SkinWindow(SkinManager manager)` | 加载当前主题、初始化列表 |
| 主题列表 | `InitializeThemeList()` | 绑定 `skinManager.Themes` |
| 主题预览 | `UpdatePreview(SkinTheme theme)` | 渐变/视频主题预览 |
| 应用主题 | `Apply_Click()` | 调用 `skinManager.ApplyTheme()` 并通知主窗口 |
| 当前主题样式 | `ApplyCurrentTheme()` | 让皮肤窗口自身跟随主题样式 |
| 待开发 | 导入动态背景 | 选择视频、复制到 `videos/`、生成主题、刷新列表 |

## 主题系统

| 文件 | 作用 |
|------|------|
| `Core/Models/SkinTheme.cs` | 主题数据模型；`BackgroundStyle` 支持 `gradient`/`solid`/`image`/`video` |
| `Core/Services/SkinManager.cs` | 管理主题列表、当前主题、应用主题、删除自定义主题 |
| `Core/Services/ThemeLoader.cs` | 从 `skins/` 读取/保存主题，生成默认暗黑主题 |
| `Core/Services/ThemeApplier.cs` | 将主题资源应用到窗口/控件，解析视频路径 |

## 工具窗口

| 文件 | 说明 |
|------|------|
| `UI/Windows/BgCropWindow.xaml(.cs)` | 背景切图工具：图片缩放、切块、写入 `cookies/u_background_pak` |
| `UI/Windows/FillBytesWindow.xaml(.cs)` | PAK 补充字节工具：读取参考字节、批量补齐、复制回游戏目录 |
| `UI/Windows/MapControl.xaml(.cs)` | 地图切换控件：扫描、预览、应用地图 |
| `UI/Windows/ConfirmDialog.xaml(.cs)` | 自定义确认弹窗 |
| `UI/Windows/TeamSelectionWindow.xaml(.cs)` | 队伍/批量选择相关窗口 |

## 关键变量 - `MainWindow.xaml.cs`

| 变量名 | 类型 | 说明 |
|--------|------|------|
| `skinManager` | `SkinManager` | 当前主题和主题列表管理器 |
| `list` | `List<ItemshopM>` | 服装道具列表 |
| `effectList` | `List<ItemshopM>` | 有特效的道具列表 |
| `beforeItems` | `List<ItemshopM>` | 变更前队列 |
| `afterItems` | `List<ItemshopM>` | 变更后队列 |
| `selectedEffect` | `ItemshopM` | 当前选中特效 |
| `strInstallDirectory` | `string` | 游戏安装目录 |
| `cookiename` | `string` | cookies 解包目录名 |
| `_libVLC` | `LibVLC?` | VLC 核心实例 |
| `_mediaPlayer` | `MediaPlayer?` | VLC 播放器 |
| `_currentMedia` | `Media?` | 当前视频媒体 |
| `_videoBitmap` | `WriteableBitmap?` | 视频帧渲染目标 |
