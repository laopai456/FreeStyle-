# FS服装搭配专家 - 代码层面操作逻辑

## 一、用户操作流程

```
步骤1: 搜索道具A名称 → 点击查看 → 设为变更前
步骤2: 搜索道具B名称 → 点击查看 → 设为变更后
步骤3: 点击变更按钮 → 确认对话框 → 执行变更
```

---

## 二、代码执行逻辑（按阶段）

### 阶段1: 点击表格查看道具

**入口**: `MainWindow.dgvItemShop_CellContentClick`

**关键变量**:
- `pakIconNum`: 图标pak文件名（从表格当前行获取）
- `pakItemNum`: 道具pak文件名（从表格当前行获取）
- `pngItemCode`: 图标文件名（u前缀 + 道具代码 + .png后缀）
- `strItemName`: 道具名称

**执行步骤**:
1. 从表格获取当前行的道具信息
2. 复制 `游戏目录\pakIconNum` → `cookies目录\pakIconNum`
3. 创建并显示 `ViewOneIconFrm2` 预览窗体

---

### 阶段2: 设为变更前

**入口**: `ViewOneIconFrm2.btnSetBefore_Click`

**调用**: `MainWindow.myRefresh(道具名, 图标文件, isbefore=true)`

**关键变量**:
- `thisIsbefore = true`: 标识正在设置变更前
- `thisOtherPropName`: 道具名称
- `thisOtherIconPng`: 图标文件名

**执行步骤**:
1. 复制 `游戏目录\pakItemNum` → `cookies目录\pakItemNum`
2. 异步执行解包: `pack\resources cookies\pakItemNum -all`
3. 解包后生成目录: `cookies\pakItemNum去点\` (包含 `bmlItemCode` 文件)
4. 完成回调设置 `tsbBeforeInfo.Text`

**tsbBeforeInfo 格式**:
```
"道具名-pak:pakItemNum,mod:bmlItemCode,eff:特效编号"
```

---

### 阶段3: 查看目标道具

**与阶段1相同**，只是获取的是目标道具的信息

---

### 阶段4: 设为变更后

**入口**: `ViewOneIconFrm2.btnSetAfter_Click`

**调用**: `MainWindow.myRefresh(道具名, 图标文件, isbefore=false)`

**关键变量**:
- `thisIsbefore = false`: 标识正在设置变更后
- `thisOtherPropName`: 道具名称
- `thisOtherIconPng`: 图标文件名
- `numEffectCode`: 特效编号（从 itemshop.txt 读取）

**执行步骤**:
1. 复制 `游戏目录\pakItemNum` → `cookies目录\pakItemNum`
2. 异步执行解包: `pack\resources cookies\pakItemNum -all`
3. 解包后生成目录: `cookies\pakItemNum去点\` (包含 `bmlItemCode` 文件)
4. 读取 `cookies\item_text_pak\itemshop.txt` 获取特效编号
5. 完成回调设置 `tsbAfterInfo.Text`

**tsbAfterInfo 格式**:
```
"道具名-pak:pakItemNum,mod:bmlItemCode,eff:特效编号,server:me/other"
```

**特效获取逻辑**:
```csharp
// 从 itemshop.txt 查找对应道具的特效编号
numEffectCode = "特效编号:XXX"  // 如果没有特效则为 "特效编号:"
// 如果 thisOtherIconPng == thisOtherPropName，说明没有特效代码
// 此时 numEffectCode 会被设置为 "无"
```

---

### 阶段5: 点击变更按钮

**入口**: `MainWindow.btnUpdate_Click`

**执行步骤**:
1. 验证 `tsbBeforeInfo` 和 `tsbAfterInfo` 是否已设置
2. 验证变更前后道具不能相同
3. 弹出确认对话框
4. 如果用户确认，启动后台线程 `bw_DoWorkLast`

---

### 阶段6: 执行核心变更逻辑

**入口**: `MainWindow.bw_DoWorkLast`

**解析信息**:
```csharp
strAfterPak = tsbAfterInfo 解析出的 pak 文件名（如 item762_pak）
strAfterMod = tsbAfterInfo 解析出的 bml 文件名（如 i50124771.bml）
strAfterEff = tsbAfterInfo 解析出的特效编号（如 "1940" 或 "无"）

strBeforePak = tsbBeforeInfo 解析出的 pak 文件名（如 item747_pak）
strBeforeMod = tsbBeforeInfo 解析出的 bml 文件名（如 i50122931.bml）
strBeforeEff = tsbBeforeInfo 解析出的特效编号（如 "无"）
```

**核心操作**:
```csharp
// 1. 复制目标模型文件到源模型文件位置
source = "cookies\strAfterPak\strAfterMod"
target = "cookies\strBeforePak\strBeforeMod"
File.Copy(source, target, true)

// 2. 重新打包源pak文件到游戏目录
Process.Start("pack\resources", "-file2pak cookies\strBeforePak 游戏目录\strBeforePak")

// 3. 处理特效
// 3.1 如果变更后有特效，变更前没有特效
if (strAfterEff != "无" && strBeforeEff == "无")
{
    // 弹出对话框询问是否添加特效
    // 如果确认，修改 itemshop.txt 添加特效
}

// 3.2 如果变更前有特效，变更后没有特效
if (strBeforeEff != "无" && strAfterEff == "无")
{
    // 弹出对话框询问是否移除特效
    // 如果确认，修改 itemshop.txt 移除特效
}

// 3.3 如果前后都有特效且不同
if (strBeforeEff != "无" && strAfterEff != "无" && strBeforeEff != strAfterEff)
{
    // 弹出对话框询问是否变更特效
    // 如果确认，修改 itemshop.txt 替换特效编号
}

// 3.4 修改 itemshop.txt 和打包
itemshopPath = "cookies\item_text_pak\itemshop.txt"
道具代码 = 从 tsbBeforeInfo 中提取
替换: 道具代码 + "\t" + strBeforePak去点 + "\t" + strBeforeEff
为:    道具代码 + "\t" + strBeforePak去点 + "\t" + strAfterEff

// 重新打包 item_text.pak
Process.Start("pack\resources", "-file2pak cookies\item_text_pak 游戏目录\item_text.pak")

// 4. 如果是它服
if (tsbAfterInfo 不包含 "server:me")
{
    // 处理 res.pak 的复制和打包逻辑
}
```

---

### 阶段7: 变更完成

**入口**: `MainWindow.bw_CompletedWorkLast`

**执行步骤**:
1. 隐藏加载动画
2. 显示成功或失败消息

---

## 三、关键数据流转

```
表格道具信息
    ↓
【查看道具】→ 复制 icon pak → cookies
    ↓
【设为变更前】→ 复制 item pak → cookies → 解包 → tsbBeforeInfo
    ↓
【查看道具】→ 复制 icon pak → cookies
    ↓
【设为变更后】→ 复制 item pak → cookies → 解包 → tsbAfterInfo
    ↓
【变更】→ 解析 tsbBeforeInfo / tsbAfterInfo
    ↓
【执行】→ 复制目标bml到源bml位置 → 打包源pak → (修改特效文本) → 完成
```

---

## 四、核心变量说明

**表格获取的变量**:
- `PakNum`: pak文件编号（如 "747"）
- `ItemCode`: 道具代码（如 "50122931"）

**派生变量**:
- `pakIconNum = "icon" + PakNum + ".pak"`
- `pakItemNum = "item" + PakNum + ".pak"`
- `pngItemCode = "u" + ItemCode + ".png"`
- `bmlItemCode = "i" + ItemCode + ".bml"`

**状态变量**:
- `tsbBeforeInfo`: 变更前道具信息字符串
- `tsbAfterInfo`: 变更后道具信息字符串
- `thisIsbefore`: 标识当前是设置变更前(true)还是变更后(false)

**目录变量**:
- `strInstallDirectory`: 游戏安装目录
- `cookiename`: 本服临时目录名（默认 "cookies"）
- `Environment.CurrentDirectory`: 程序运行目录

---

## 五、关键方法映射

| 阶段 | 触发方法 | 执行方法 |
|------|----------|----------|
| 查看道具 | `dgvItemShop_CellContentClick` | 复制 icon pak |
| 设为变更前 | `btnSetBefore_Click` → `myRefresh` | `bw_DoWorkOtherItem` → `bw_CompletedWorkOtherItem` |
| 设为变更后 | `btnSetAfter_Click` → `myRefresh` | `bw_DoWorkOtherItem` → `bw_CompletedWorkOtherItem` |
| 执行变更 | `btnUpdate_Click` | `bw_DoWorkLast` → `bw_CompletedWorkLast` |

---

## 六、核心命令

**解包命令**:
```
pack\resources cookies\pakItemNum -all
```

**打包命令**:
```
pack\resources -file2pak cookies\pakItemNum去点 游戏目录\pakItemNum
```

**特效文本打包**:
```
pack\resources -file2pak cookies\item_text_pak 游戏目录\item_text.pak
```
