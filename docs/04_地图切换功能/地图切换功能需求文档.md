# 地图切换功能需求文档

## 一、功能概述

将现有的"角色语音"功能改造为"地图切换"功能，用于快速切换街头篮球游戏的游戏地图。

---

## 二、用户操作流程

```
步骤1: 打开"地图切换"标签页
步骤2: 设置地图库路径（首次使用或需要更改时）
步骤3: 系统自动扫描地图库目录，加载所有可用地图
步骤4: 在左侧表格中浏览地图列表
步骤5: 点击地图名称 → 右侧显示预览图
步骤6: 点击"应用"按钮 → 将选中的地图覆盖到游戏目录
```

## 五、功能逻辑详解

### 5.1 地图库路径配置

**触发时机：**
- 控件加载时（`MapControl_Load`）- 加载上次保存的路径
- 点击"浏览..."按钮（`btnBrowseLibrary_Click`）- 选择新的地图库路径
- 点击"刷新"按钮（`btnRefreshLibrary_Click`）- 使用当前路径重新扫描

**路径保存：**
- 保存到配置文件：`Settings.Default.MapLibraryPath`
- 或保存到 INI 文件：`map_config.ini`
- 格式：`MapLibraryPath=C:\Users\w\Desktop\fs\颜值地图\第一期\`

**代码逻辑：**
```csharp
// 加载上次保存的路径
private void MapControl_Load(object sender, EventArgs e)
{
    string savedPath = Settings.Default.MapLibraryPath;

    if (string.IsNullOrEmpty(savedPath))
    {
        // 默认路径
        savedPath = @"C:\Users\w\Desktop\fs\颜值地图\第一期\";
    }

    txtMapLibraryPath.Text = savedPath;
    LoadMapsFromLibrary(savedPath);
}

// 浏览选择文件夹
private void btnBrowseLibrary_Click(object sender, EventArgs e)
{
    using (FolderBrowserDialog dialog = new FolderBrowserDialog())
    {
        dialog.Description = "选择地图库文件夹";
        dialog.SelectedPath = txtMapLibraryPath.Text;

        if (dialog.ShowDialog() == DialogResult.OK)
        {
            txtMapLibraryPath.Text = dialog.SelectedPath;
            Settings.Default.MapLibraryPath = dialog.SelectedPath;
            Settings.Default.Save();

            LoadMapsFromLibrary(dialog.SelectedPath);
        }
    }
}

// 刷新地图列表
private void btnRefreshLibrary_Click(object sender, EventArgs e)
{
    string libraryPath = txtMapLibraryPath.Text;

    if (!Directory.Exists(libraryPath))
    {
        MessageBox.Show("地图库路径不存在，请重新选择！", "错误");
        return;
    }

    LoadMapsFromLibrary(libraryPath);
}
```

### 5.2 扫描地图库

**触发时机：**
- 地图库路径变化时（加载、浏览、刷新）

**执行步骤：**
1. 读取地图库目录（从 `txtMapLibraryPath.Text`）
2. 验证目录是否存在
3. 遍历所有子文件夹
4. 对每个文件夹：
   - 检查是否存在 `stage02.pak` 文件
   - 查找第一个 `.png` 或 `.jpg` 文件作为预览图
   - 构建 `MapM` 对象
5. 将所有 `MapM` 添加到表格
6. 更新加载状态标签

**代码逻辑：**
```csharp
private void LoadMapsFromLibrary(string mapLibraryPath)
{
    if (!Directory.Exists(mapLibraryPath))
    {
        MessageBox.Show("地图库路径不存在，请重新选择！", "错误");
        return;
    }

    List<MapM> maps = new List<MapM>();

    foreach (string folder in Directory.GetDirectories(mapLibraryPath))
    {
        string mapName = Path.GetFileName(folder);
        string pakPath = Path.Combine(folder, "stage02.pak");

        // 检查 stage02.pak 是否存在
        if (!File.Exists(pakPath))
            continue;

        // 查找预览图（第一个 png 或 jpg）
        string previewPath = Directory.GetFiles(folder, "*.png")
            .Concat(Directory.GetFiles(folder, "*.jpg"))
            .FirstOrDefault();

        if (previewPath == null)
            continue;

        maps.Add(new MapM
        {
            MapName = mapName,
            PreviewPath = previewPath,
            PakPath = pakPath,
            FolderPath = folder
        });
    }

    // 绑定到表格
    allMaps = maps; // 保存原始列表用于搜索
    dgvMap.DataSource = maps;

    lblStatus.Text = $"已加载 {maps.Count} 个地图";
}
```

### 5.3 预览功能

**触发时机：**
- 用户点击表格中的某一行（`dgvMap_CellClick`）

**执行步骤：**
1. 获取选中行的地图信息
2. 加载完整预览图到右侧 PictureBox
3. 更新地图名称和路径标签
4. 启用"应用"按钮

**代码逻辑：**
```csharp
private void dgvMap_CellClick(object sender, DataGridViewCellEventArgs e)
{
    if (e.RowIndex < 0) return;

    DataGridViewRow row = dgvMap.Rows[e.RowIndex];
    string mapName = row.Cells["MapName"].Value.ToString();
    string previewPath = row.Cells["FolderPath"].Value.ToString();
    string pakPath = row.Cells["PakPath"].Value.ToString();

    // 查找预览图
    string[] previewFiles = Directory.GetFiles(previewPath, "*.png")
        .Concat(Directory.GetFiles(previewPath, "*.jpg"))
        .ToArray();

    if (previewFiles.Length > 0)
    {
        picPreview.Image = Image.FromFile(previewFiles[0]);
        picPreview.SizeMode = PictureBoxSizeMode.Zoom;
    }

    lblMapName.Text = mapName;
    lblMapPath.Text = pakPath;
    btnApply.Enabled = true;

    selectedMap = new MapM
    {
        MapName = mapName,
        PreviewPath = previewFiles.Length > 0 ? previewFiles[0] : "",
        PakPath = pakPath,
        FolderPath = previewPath
    };
}
```

### 5.5 应用地图

**触发时机：**
- 用户点击"应用"按钮（`btnApply_Click`）

**执行步骤：**
1. 验证是否选中了地图
2. 弹出确认对话框："确认应用地图：{地图名称}？"
3. 如果用户确认：
   - 获取游戏目录：`strInstallDirectory`（复用 MainWindow 的变量）
   - 目标路径：`{游戏目录}\stage02.pak`
   - 源路径：`{选中地图文件夹}\stage02.pak`
   - 复制文件（覆盖模式）
   - 显示成功消息
4. 如果用户取消，不做操作

**代码逻辑：**
```csharp
private void btnApply_Click(object sender, EventArgs e)
{
    if (selectedMap == null)
    {
        MessageBox.Show("请先选择地图", "提示");
        return;
    }

    DialogResult result = MessageBox.Show(
        $"确认应用地图：{selectedMap.MapName}？\n\n" +
        $"目标路径：{strInstallDirectory}\\stage02.pak",
        "确认",
        MessageBoxButtons.YesNo,
        MessageBoxIcon.Question
    );

    if (result == DialogResult.Yes)
    {
        try
        {
            string targetPath = Path.Combine(strInstallDirectory, "stage02.pak");

            // 复制文件
            File.Copy(selectedMap.PakPath, targetPath, true);

            MessageBox.Show("地图应用成功！", "成功");
        }
        catch (Exception ex)
        {
            MessageBox.Show($"应用失败：{ex.Message}", "错误");
        }
    }
}
```

## 八、注意事项

### 8.1 预览图识别规则
- 优先识别第一个 `.png` 文件
- 如果没有 `.png`，识别第一个 `.jpg` 文件
- 文件夹内只保留一个预览图（确保唯一性）

### 8.2 游戏目录获取
- 复用 `MainWindow.strInstallDirectory` 变量
- 如果该变量为空，提示用户先设置游戏目录

### 8.3 地图库路径保存
- 保存到 `Settings.Default.MapLibraryPath`
- 或保存到 `map_config.ini` 文件
- 首次使用时显示默认路径提示用户确认

### 8.3 文件复制安全
- 使用 `File.Copy(source, target, true)` 允许覆盖
- 添加异常处理（文件占用、权限不足等）
- 提示用户关闭游戏后再操作

### 8.4 性能优化
- 缩略图异步加载（避免表格卡顿）
- 预览图缓存（避免重复加载）
