# 背景修改功能开发进度

## 功能概述
在常用工具菜单中新增"背景修改"功能，用于将用户上传的图片切分成街头篮球背景图片所需的尺寸，并自动保存到解压目录。

## 项目位置
- 主程序: `d:/py/反编译/FS服装搭配专家v1.0/`
- 源代码: `FS服装搭配专家v1.0.csproj`

## 功能需求

### 用户操作流程
```
1. 点击"常用工具"菜单 → "背景修改"
2. 在切图窗口中点击"浏览..."按钮
3. 选择一张图片文件
4. 从下拉框中选择背景类型（登录界面、角色选择、大厅背景等）
5. 点击"开始修改"按钮
6. 程序自动解压 u_background.pak → 缩放图片 → 切图替换 → 完成
7. 使用"补充字节"功能重新打包 pak 文件（见补充字节功能）
```

### 技术实现

#### 1. 切图逻辑
- **缩放**: 将输入图片缩放到 1366x768
- **切割**: 按 512x512 切块，生成 3x3 = 6张图片
- **特殊处理**: 将第4张和第5张图片垂直合并成一张（512x512）
- **透明背景**: 使用 Format32bppArgb + NearestNeighbor 避免白边
- **命名规则**:
  - bg1027_0.png, bg1027_1.png, bg1027_2.png
  - bg1027_3.png (合并后的第4张)
- **保存位置**: `cookies\u_background_pak\`

#### 2. 背景类型映射
| 序号 | 背景名称 | 文件前缀 | 说明 |
|------|---------|---------|------|
| 0 | 登录界面 | bg1027 | 游戏登录背景 |
| 1 | 角色选择 | bg1013 | 创建角色界面 |
| 2 | 大厅背景 | bg1012 | 游戏大厅 |
| 3 | 等待房间 | bg1014 | 房间等待界面 |
| 4 | 读取加载 | bg1004 | 加载画面1 |
| 5 | 读取加载 | bg1005 | 加载画面2 |
| 6 | 读取加载 | bg1006 | 加载画面3 |

#### 3. 操作流程
```
开始修改
  ↓
检查游戏路径
  ↓
解压 u_background.pak → cookies\u_background_pak (首次使用)
  ↓
缩放输入图片到 1366x768
  ↓
切割成 512x512 图片块
  ↓
合并第4、5张图片
  ↓
按命名规则保存到 cookies\u_background_pak
  ↓
显示成功/失败提示
  ↓
（可选）使用补充字节功能重新打包 pak
```

#### 4. 路径配置
- **源pak文件**: `游戏目录\u_background.pak`
- **解压目录**: `cookies\u_background_pak`
- **解压工具**: `pack\resources` (与服装道具相同)
- **解压命令**: `pack\resources cookies\u_background_pak -all`

#### 图片切边有白边问题
- **问题**: 合并后的图片边缘有白边

## 补充字节功能

### 功能概述
pak 文件解包后重新打包会导致文件大小变化，需要补充字节使新文件大小与源文件一致，否则游戏会报错。

### 用户操作流程
```
1. 点击"常用工具"菜单 → "补充字节"
2. 查看最近使用的 pak 文件（最多3个）
3. 或在搜索框中输入文件名过滤查找
4. 勾选需要补充字节的文件
5. 点击"补充选中文件"按钮
6. 程序自动补充字节并更新文件
```

### 技术实现

#### 1. 补充字节逻辑
```csharp
// 伪代码示例
void FillBytes(string pakFilePath)
{
    // 1. 读取源 pak 文件大小
    long originalSize = new FileInfo(sourcePakPath).Length;

    // 2. 读取新打包的文件大小
    long newSize = new FileInfo(newPakPath).Length;

    // 3. 如果新文件小于源文件，补充字节到源文件大小
    if (newSize < originalSize)
    {
        using (FileStream fs = new FileStream(newPakPath, FileMode.Append))
        {
            byte[] padding = new byte[originalSize - newSize];
            fs.Write(padding, 0, padding.Length);
        }
    }

    // 4. 更新最近使用记录
    UpdateRecentFiles(pakFilePath);
}
```

#### 2. 最近使用持久化
```csharp
// 保存到配置文件
string recentFilesConfig = "u_background.pak|icon.pak|item.pak";

// 加载配置
string[] recentFiles = recentFilesConfig.Split('|');
```

#### 3. 文件扫描逻辑
```csharp
// 扫描游戏目录下所有 pak 文件
List<string> ScanPakFiles()
{
    string gameDir = parentForm.strInstallDirectory;
    return Directory.GetFiles(gameDir, "*.pak", SearchOption.TopDirectoryOnly)
                    .OrderBy(f => Path.GetFileName(f))
                    .ToList();
}
```
