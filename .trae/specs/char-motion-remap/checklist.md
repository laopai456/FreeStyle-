# CharacterMotion Type Remap — 验证清单

## 阶段 1 验证：Type 字段定位

- [ ] ItemCode 25461 在堆上有 ≥1 个命中
- [ ] 至少一个命中附近 ±0x200 内找到 0x0C (Static)
- [ ] ItemCode 50125031 (动态) 附近找到 0x0B (Dynamic)
- [ ] 两个发型的 Type 偏移量一致
- [ ] Type 字段所在对象有合法 vtable

## 阶段 2 验证：Type 修改效果

### 安全性验证

- [ ] Type 修改后 APOLLO 未杀进程
- [ ] 修改值能被回读确认（写入成功）
- [ ] Type 值在 15 秒内未被自动覆盖（或覆盖后能自动重写）

### 显示验证

- [ ] 重新装备后发型正常显示（非光头）
- [ ] 发型纹理正确（非黑色/错位）
- [ ] 角色身体其他部分正常

### 物理验证

- [ ] 角色站立时头发有自然下垂
- [ ] 角色移动/跳跃时头发有摆动
- [ ] 摆动效果与原生动态发型相似
- [ ] 物理效果持续存在（不消失）

### 技术验证

- [ ] verify_bone_count.py 显示骨骼数变化（61→74+）
- [ ] full_dump.py 显示物理参数已自动填充（非手动写入）
- [ ] DSkeleton +0x01C = 24（物理骨骼数自动设置）
- [ ] DSkeleton +0x020 ≈ 0.9583（刚度自动设置）
- [ ] DSkeleton +0x02C ≈ 1.25（重力自动设置）

## 回滚方案

如果测试失败：

- [ ] Type 字段改回 0x0C (Static)
- [ ] 恢复 res767.pak.bak
- [ ] 重启游戏确认恢复正常

## 成功标准

全部满足即视为成功：

1. APOLLO 不杀进程
2. 发型显示正常
3. 物理摆动效果与原生动态发型一致
4. 效果在 5 分钟内持续稳定
